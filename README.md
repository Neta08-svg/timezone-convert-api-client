# Free Timezone Conversion API

Free timezone and timestamp conversion API - historically and DST-correct conversions between IANA zones.

**Live tool & docs:** https://tz.wrapper-agency.com

## Quickstart - free API, no key required

```bash
curl "https://tz.wrapper-agency.com/api/v1/tz?from=America/New_York&to=Asia/Tokyo&time=2008-11-04T09:00"
```

Clean JSON, open CORS, no signup. Try the interactive tool and full docs at https://tz.wrapper-agency.com
